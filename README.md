# WEB103 Prework - *Creatorverse*

Submitted by: **Tanvir Pulak**

About this web app: **Creatorverse is a full-stack React web application that allows users to discover, add, edit, view, and delete their favorite content creators. The app uses Supabase as a backend database and React Router for navigation, enabling full CRUD functionality for creators across platforms such as YouTube, Twitch, TikTok, and Instagram.**
